### 2.2 - Editores de código (IDE) compatíveis

Para manter um padrão de código a ser seguido, é recomendado o editor de código Visual Studio Code na mais atual (Versão: 1.95.3 de 2024-11-13), com algumas extensões que permitem Linting, formatação automática e outras facilidades.

Cada versão do projeto, por padrão, mantém atualizado no repositório o arquivo .vscode/settings.json gerado pelo VSCode. Esse arquivo permite que configurações de formatação de código e de extensões sejam compartilhadas entre os desenvolvedores, ou seja, no âmbito de workspace.

Assim, através do uso de extensões e compartilhamento da configuração do workspace, padroniza-se o uso de indentação, ponto-e-vírgula, quebra de linha, etc., o que reduz consideravelmente o atrito no momento de fazer merge de código.
