#### 2.3.2 – Javascript (JS) ES6+

A linguagem principal utilizada neste projeto foi Javascript (JS).

O JavaScript, em suas revisões mais recentes (ES6+) foi a linguagem escolhida para desenvolver o Frontend e o Backend e seus subprojetos (Ex: App), aplicando tecnologias que possibilitam o uso da linguagem para construir e rodar a aplicação nos ambientes desejados, como o Node.js e o VueJS.

Para executar o código em ambiente de desenvolvimento e fazer o deployment, basta seguir as instruções que estão no README de cada projeto. Estamos seguindo os padrões de indústria para esses procedimentos, portanto, devem ser intuitivos para qualquer desenvolvedor que conheça essas tecnologias.
